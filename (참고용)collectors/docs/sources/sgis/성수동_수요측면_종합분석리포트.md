# 성수동 수요측면 종합 분석 리포트  
**크래프톤 사옥 성수동 이전에 따른 1인가구(20~49세) 주거지원 정책 — 수요 분포 현황**

**배경**: 크래프톤이 **사옥을 성수동으로 이전**하며, **1인가구(20~49세) 주거지원**을 위해 **동일 대상**의 수요 분포를 확인하고 **정책에 반영**하고자 함.  
**목적**: 이전지(성수동) 내 **20~49세 1인가구**의 규모·연령·거주 유형·연면적·동별 분포를 파악하여, **주거지원 정책 수립·검토 시 정책 설계에 고려**할 수 있는 기초 자료를 제공함.  
**분석 대상**: 서울 성동구 성수동 (성수1가1동, 성수1가2동, 성수2가1동, 성수2가3동)  
**집계 단위**: 집계구(tract) 108개  
**데이터 출처**: `sgis_fusion_1person_20_49_20260203_033203` (20~49세 1인가구 결합조건 수집)  
**기준 연도**: 2025년 경계, 통계 연도는 수집 시점 기준

---

## 1. 개요

크래프톤은 **사옥을 성수동으로 이전**하고, **1인가구(20~49세) 주거지원** 정책을 수립·운영하고자 한다. 본 리포트는 **이전지(성수동) 내 동일 대상(20~49세 1인가구)** 의 **수요 분포**를 SGIS 결합조건(Fusion) API 직접 집계 데이터로 파악하여, **주거지원 정책 설계 시 정책에 고려**할 수 있도록 구성했다.  
- **수요 분포**: 인구·가구 규모, 연령대별, 주택유형(ht), 연면적(ha), 동별 분포  
- **정책 활용**: 대상 규모 산정, 지역·연령·주거 유형별 우선순위, 지원 형태(임대·개선·시설 등) 설계 시 참고

| 항목 | 내용 |
|------|------|
| **정책 대상** | 20~49세 1인가구(주거지원 정책의 동일 대상) |
| **수요 분포 지역** | 성수동 4개 동, 108개 집계구(사옥 이전지와 동일) |
| **지표** | 인구·가구수, 연령대별, 주택유형(ht), 연면적(ha) |

### 1.1 데이터 신뢰도

본 데이터는 **20~49세·1인가구(A0)** 조건으로 API에서 직접 집계한 값이다.  
- **ht01~ht05, ha01~ha09**: API 주택유형·연면적 코드표에 따른 **직접 집계값** → 결론 도출에 **가장 확실**하게 활용할 수 있다.  
- **ht07·ha10**: API 코드표에 없는 값으로, 수집 스크립트에서 **오피스텔 추정**으로 산출. **참고용**으로만 볼 것.

## 6. 사옥 이전에 따른 주거지원 정책 — 수요 분포 기반 정책 고려사항

사옥 **성수동 이전**에 따라 **1인가구(20~49세) 주거지원** 정책을 수립·운영할 때, 위 **동일 대상의 수요 분포**를 반영한 정책 고려사항을 정리한다.

### 6.1 수요 분포 요약 — 정책 설계 시 반영

| 항목 | 수요 분포 내용 | 정책에 고려할 시점 |
|------|----------------|---------------------|
| **대상 규모** | 이전지(성수동) 내 20~49세 1인가구 **약 6,200명·6,280가구** | 주거지원 대상 규모·예산·사업 규모 산정 시 상한 참고 |
| **연령 구성** | 30대(45.5%) > 20대(34.6%) > 40대(19.9%) | 연령대별 우선 지원(30대·20대 중심) 검토 시 정책 반영 |
| **거주 유형** | 단독 41.6%, 다세대 15.8%, 아파트 13.5%. **78% 이상이 일반 주택** | **단독·다세대·아파트** 거주층을 핵심 대상으로 한 지원 유형 설계에 부합 |
| **동별 분포** | 성수1가2동·성수1가1동 규모 약간 큼, 성수2가3동 상대적 소규모 | 지역 배분·우선 시범 동 설정 시 정책에 고려 |
| **동별 주거 유형** | 성수2가1동 단독·일반주택 비중 높음, 성수1가1동 아파트 비중 높음 | **주거 유형 연계 지원** 시 동별 차등 정책 검토 |
| **세부 지역** | 집계구 108개 | 동 내 **세부 타깃 구간**·시설 입지·홍보 구간 선정 시 정책 반영 |

### 6.2 주거지원 정책 수립·검토 시 정책 고려사항

1. **대상 규모**  
   이전지(성수동) 내 동일 대상 **6,200명·6,280가구**를 상한 참고치로 두고, 지원 형태(전수·일부·신청자 등)에 따라 **정책 대상 규모**를 설정할 수 있다.

2. **연령·지역·주거 유형**  
   - **연령**: 30대·20대 비중이 크므로, 청년 1인가구 주거지원 강조 시 위 연령대를 우선 대상으로 두는 정책 검토 가능.  
   - **지역**: 동별 수요 비중(성수1가2동·성수1가1동 25~26%)을 반영한 **동별 배분** 또는 **특정 동 우선 시범**을 정책에 고려할 수 있다.  
   - **주거 유형**: **단독·다세대·아파트** 거주층이 78% 이상이므로, 주거 개선·임대·시설 이용 등 **일반 주택 거주자 중심** 지원이 수요 분포와 맞다.

3. **세부 타깃팅**  
   집계구(108개) 단위 데이터로 **동 내 수요 집중 구간**을 파악하면, 시설 입지·모집 지역·홍보 구간 선정 등 **정책 실행 단위** 설계에 반영할 수 있다.

### 6.3 참고 사항 (추정값)

- **ht07·ha10**(오피스텔 추정)은 API 코드표에 없는 **추정값**이므로, 오피스텔 비중·동별 오피스텔 차이는 **참고 수준**으로만 정책에 활용할 것.
