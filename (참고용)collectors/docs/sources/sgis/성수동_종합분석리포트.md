## 5. 종합 해석 및 시사점

### 5.1 확실한 결론 (1인가구 수요 데이터 기반)

1. **20~49세 1인가구 규모**  
   성수동 전체 가구의 **26.3%**(6,284가구), 20~49세 인구의 **28.2%**가 20~49세 1인가구다. API 직접 집계로 확인 가능하다.

2. **20~49세 1인가구의 거주 유형**  
   이 수요층은 **단독주택(ht01) 41.6%**, **다세대(ht04) 15.8%**, **아파트(ht02) 13.5%** 순으로 거주한다. **ht01~ht05만으로 약 78%**를 차지하며, **단독·다세대·아파트에 집중**되어 있다는 것은 확실히 말할 수 있다.

3. **연령 구성**  
   30대(45.5%), 20대(34.6%), 40대(19.9%) 순으로, 30대 비중이 가장 크다.

4. **정책·사업 시**  
   20~49세 1인가구 대상 주거 지원·정책은 **단독·다세대·아파트 거주층**을 함께 고려하는 것이 타당하다. 집계구(108개) 단위로 수요 지표를 매핑하면 **동 내 세부 지역별** 수요 집중 구간 파악에 활용할 수 있다.

### 5.2 참고 시사점 (오피스텔 추정값 전제)

- **공급 측 ht07·ha10**은 API 코드표에 없는 **추정값**이다. “성수동 주택 재고의 64%가 오피스텔”이라는 해석은 이 추정에 의존하므로 **미심쩍을 수 있으며, 참고 수준**으로 보는 것이 좋다.
