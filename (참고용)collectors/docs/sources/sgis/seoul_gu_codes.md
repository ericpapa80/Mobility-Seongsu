# 서울시 구 행정구역 코드 참고

## 문제 상황

SGIS API에서 행정구역 코드와 실제 반환되는 데이터가 일치하지 않는 경우가 있습니다.

## 확인된 코드 매핑

### 테스트 결과

- `11200`: **동작구** 데이터 반환 (성동구가 아님)
- `11210`: **관악구** 데이터 반환
- `11560`: 검색 결과 없음

## 해결 방법

### 1. SGIS 웹사이트에서 확인

SGIS 웹사이트(https://sgis.mods.go.kr/view/technicalBiz/technicalBizMap?tec=0)에서:
1. 성동구를 선택
2. 브라우저 개발자 도구(F12) 열기
3. Network 탭에서 API 요청 확인
4. `adm_cd` 파라미터 값 확인

### 2. 수집된 데이터에서 역추적

이미 수집된 데이터에서 성동구 주소가 포함된 항목을 찾아 해당 `adm_cd`를 확인할 수 있습니다.

### 3. 행정구역 코드 매핑 테이블 구축

각 구의 올바른 코드를 찾아 매핑 테이블을 만들어 관리하는 것을 권장합니다.

## 참고

- 표준 행정구역 코드와 SGIS API 코드가 다를 수 있습니다
- 정확한 코드는 SGIS 공식 문서나 실제 API 테스트를 통해 확인해야 합니다
