2. **페이지 20, 37, 73**: theme_cd=5001 예제

### 5. 결론

1. **일반 테마 코드표**: PDF에 명시적으로 제공됨 (대분류 A-K, 소분류 4자리 숫자)
2. **기술업종 코드표**: PDF에 제공됨 (techbiz_cd: 11-23)
3. **기술업종 통계지도 theme_cd**: PDF에 명시적인 정의가 없음
   - `getPoiCompanyDensity` API는 PDF에 포함되지 않은 것으로 보임
   - 실제 수집 데이터의 theme_cd 값(110, 111, 112 등)은 PDF의 일반 테마 코드표와 다른 체계

### 6. 권장 사항

1. **SGIS 개발자 지원센터 직접 문의**
   - URL: https://sgis.mods.go.kr/developer
   - 기술업종 통계지도 API의 theme_cd 코드표 요청

2. **웹사이트에서 확인**
   - 기술업종 통계지도: https://sgis.mods.go.kr/view/technicalBiz/technicalBizMap?tec=0
   - 브라우저 개발자 도구로 API 요청 확인

3. **실제 데이터 분석**
   - 수집된 데이터의 theme_cd 값과 회사명을 분석하여 패턴 파악
   - 이미 `THEME_CD_INDUSTRY_COMPARISON.md`에 일부 분석 결과 있음

## 참고

- PDF의 일반 테마 코드표는 일반 사업체 정보 API용으로 보임
- 기술업종 통계지도는 별도의 코드 체계를 사용하는 것으로 추정됨
- `theme_cd=0`은 "전체 기술업종"을 의미하는 것으로 확인됨
