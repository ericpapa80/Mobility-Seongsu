## 데이터 제한사항

1. **상한액 적용**: 국민연금법 시행령 제5조에 의거 기준소득월액 상한액이 적용되어 실제 소득과 고지금액은 상이할 수 있습니다.
   - 2019.7.~2020.6.: 4,860,000원
   - 2020.7.~2021.6.: 5,030,000원
   - 2021.7.~2022.6.: 5,240,000원

2. **신규/상실자수**: 전달 고지대상자와 비교하므로 실제 취득/퇴사자와 상이할 수 있습니다.

3. **가입 제외 대상**: 
   - 60세 도달자
   - 퇴직연금수급자
   - 조기노령연금 수급권 취득자
   - 18세 미만
   - 기초수급자 (본인희망 시)
