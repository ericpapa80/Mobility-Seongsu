# OpenUp 성수동 매장 데이터 확장 CSV 메타데이터

## 개요

이 문서는 `openup_seongsu_stores_{timestamp}_expanded.csv` 파일의 컬럼 구조와 각 필드의 의미를 설명합니다.

이 CSV 파일은 원본 JSON 데이터에서 배열 필드들을 BI 툴(Power BI, Tableau, Excel 등)에서 활용하기 쉽도록 개별 컬럼으로 분리한 형태입니다.

## 파일 정보

- **파일명**: `openup_seongsu_stores_{timestamp}_expanded.csv`
- **원본 파일**: `openup_seongsu_stores_{timestamp}.json`
- **변환 스크립트**: `collectors/scripts/openup/convert_json_to_expanded_csv.py`
- **인코딩**: UTF-8 with BOM (utf-8-sig)

## 컬럼 구조

총 **44개 컬럼**으로 구성되어 있으며, 기본 정보 필드와 배열 데이터가 개별 컬럼으로 분리된 매출 데이터로 구성됩니다.

## 주요 특징

### 1. 배열 데이터의 개별 컬럼 분리

원본 JSON 파일에서는 배열 형태로 저장되어 있던 데이터들이 이 CSV에서는 각 요소가 개별 컬럼으로 분리되어 있어, BI 툴에서 필터링, 그룹화, 집계 작업을 쉽게 수행할 수 있습니다.

### 2. BI 툴 친화적 구조

- 각 매출 데이터가 개별 컬럼으로 분리되어 있어 Excel, Power BI, Tableau 등에서 바로 활용 가능
- 컬럼명에 의미가 포함되어 있어 가독성 향상
- 숫자형 데이터는 적절한 타입(int/float)으로 저장

### 3. 제외된 필드

원본 JSON의 `trend` 필드는 시계열 배열 데이터로 구성되어 있어 별도 처리 없이 이 CSV에서는 제외되었습니다. 시계열 분석이 필요한 경우 원본 JSON 파일을 참조하시기 바랍니다.

## 사용 예시

### Excel/Power BI에서 활용

1. **필터링**: 특정 연령대나 시간대 매출이 높은 매장 찾기
   - 예: `gender_f_20대` > 100 AND `times_점심` > 500

2. **그룹화**: 카테고리별 평균 매출 비교
   - 예: `category_bg`별 `times_점심` 평균

3. **집계**: 요일별 총 매출 계산
   - 예: `weekday_월` + `weekday_화` + ... + `weekday_일`

4. **시각화**: 
   - 시간대별 매출 막대 그래프 (`times_*` 컬럼들)
   - 연령대별 매출 파이 차트 (`gender_f_*`, `gender_m_*` 컬럼들)
   - 요일별 매출 라인 차트 (`weekday_*` 컬럼들)
