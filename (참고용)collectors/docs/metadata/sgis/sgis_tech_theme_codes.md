# SGIS 테마 코드(theme_cd) 설명

## 개요

`theme_cd`는 SGIS(통계지리정보서비스) 기술업종 통계지도에서 사용하는 테마 코드입니다.

## 사용 방식

### 요청 파라미터

API 요청 시 `theme_cd` 파라미터를 사용합니다:
- `theme_cd=0`: 전체 기술업종 (기본값)
- 다른 값: 특정 업종 필터링

### 응답 데이터

수집된 데이터의 각 항목에도 `theme_cd` 필드가 포함되어 있으며, 이는 해당 회사/POI가 속한 업종 테마를 나타냅니다.

## PDF 정의서에서 확인한 내용

`SGIS_OpenAPI_정의서.pdf`에서 확인한 내용:

### 1. 일반 테마 코드표 (일반 API용)

PDF에 명시된 일반 테마 코드표:
- **대분류**: A(농림어업), B(기업), C(소매업), D(생활서비스), E(교통), F(여가생활), G(숙박), H(음식), I(교육), J(의료), K(공공), 99(기타)
- **소분류**: 1001(인테리어), 5001(한식), 8001(제조/화학) 등 4자리 숫자

### 2. 기술업종 코드표 (기술업종 API용)

페이지 113에서 확인:
- 11: 첨단기술
- 12: 고기술
- 13: 중기술
- 14: 저기술
- 21: 창의 및 디지털
- 22: ICT
- 23: 전문서비스

**참고**: 기술업종 API에서는 `techbiz_cd` 파라미터를 사용합니다.

### 3. 기술업종 통계지도 API의 theme_cd

**중요**: PDF에서 `getPoiCompanyDensity` API의 `theme_cd`에 대한 명시적인 정의를 찾을 수 없었습니다.

- PDF에는 일반 테마 코드표와 기술업종 코드표만 포함되어 있음
- 실제 수집 데이터의 theme_cd 값(110, 111, 112 등)은 PDF의 코드표와 다른 체계로 보임
- 기술업종 통계지도는 별도의 코드 체계를 사용하는 것으로 추정됨

**✅ 해결**: `sgis_theme_codes_sub.md` 파일에서 기술업종 통계지도 API의 theme_cd 코드표를 확인했습니다. (위의 "기술업종 통계지도 theme_cd 코드표" 섹션 참조)

## 테마 코드 확인 방법

1. SGIS 웹사이트에서 기술업종 통계지도 접속
2. 다양한 업종 필터 선택
3. 브라우저 개발자 도구에서 API 요청 확인
4. `theme_cd` 파라미터 값 확인
