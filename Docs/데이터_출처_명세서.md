# Mobility-Seongsu 데이터 출처 명세서

각 출처의 제공처, 갱신 주기, 제공 방식, 유무료, 단위, 활용, 대상 등을 정리한 명세서입니다.

## 클러스터 (워크스페이스별)

| 클러스터 | 워크스페이스 | 포함 항목 |
|----------|--------------|-----------|
| **유동 흐름** | Dynamic Flow | 버스, 지하철, 교통속도, 보행자, 따릉이, 실시간 인구·도로소통·지하철·버스·따릉이 도착 |
| **상권** | Commercial | 상가, 사업장/급여, 건물, 상권경계, 크래프톤, 장소혼잡도, 실시간 상권 |
| **인프라** | Infrastructure | 도로·보도(정밀도로지도), 횡단보도, 어린이보호구역, 건축 인허가, S-Map 3D 건물, 전기차충전소, 주차장 |
| **안전** | Risk & Safety | 위험 구간, 사고통제현황, 날씨, 문화행사 |

## 1. 버스 정류장

| 항목 | 내용 |
|------|------|

| **활용** | 버스 승하차 패턴, 정류장별 이용량, 교통카드 기반 분석 |

| **참고** | 1단계: 공공데이터포털 15000303 정류소정보조회, 2단계: CardBusTimeNew |
