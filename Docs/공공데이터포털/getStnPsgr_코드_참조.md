# getStnPsgr API 코드 참조

## trnscdSeCd / trnscdSeCdNm (교통카드 구분)

| 코드 | 명칭 | 의미 |
|------|------|------|
| 1 | 선불카드 | 선불 방식 결제 (티머니, 캐시비 등 충전형) |
| 2 | 후불카드 | 신용·체크카드 등 후불 결제 |
| B | 정기권카드 | 지하철 정기권 |
| C, C1 | 1회용교통카드 | 편의점·역 매표기 등 1회용 교통카드 |
