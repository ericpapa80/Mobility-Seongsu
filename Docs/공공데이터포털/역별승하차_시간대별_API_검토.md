## 3. getStnPsgr (B553766) - 기존 프로젝트 사용 API

| 항목 | 내용 |
|------|------|
| **URL** | `https://apis.data.go.kr/B553766/psgr/getStnPsgr` |
| **제공** | 공공데이터활용지원센터 또는 지하철청 계열 |

### 파라미터 (참고: 외부 문서 기준)
- `serviceKey`: 인증키
- `pasngYmd`: 통행일자 (YYYYMMDD)
- `stnNm`: 역명
- `lineNm`: 노선
- `pasngHr`: **통행시간 (00-23)** ← 시간대별 조회 지원 가능
- `stnCd`: 역코드
- `numOfRows`, `pageNo` 등

### 특징
- 시간대별 파라미터 `pasngHr` 존재 (일부 문서)
- 호출 시 500 에러 발생 → 서비스 상태·날짜 제한 재확인 필요
- 서울숲역 (수인분당선) 지원 여부 불명
