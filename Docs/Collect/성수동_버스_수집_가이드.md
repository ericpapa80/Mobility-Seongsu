# 성수동 버스정류장 24시간 승하차 수집 가이드

## 개요

성수동에 위치한 모든 버스정류장의 시간대별(00~23시) 승하차 인원을 수집하는 2단계 절차입니다.

| 단계 | 목적 | API |
|------|------|-----|
| 1단계 | 성수동 정류장 + 경유노선 목록 확보 | 서울시 버스 API (ws.bus.go.kr) |
| 2단계 | 해당 노선별 시간대별 승하차 수집 | 서울열린데이터 CardBusTimeNew |

## 1단계: 정류장·경유노선 목록

### 실행

```bash
python Collect/fetch_seongsu_bus_stops.py
```

### 필요 조건

- 공공데이터포털 [서울특별시_정류소정보조회(15000303)](https://www.data.go.kr/data/15000303/openapi.do) 활용신청
- `.env`에 `SEOUL_BUS_API_KEY` 또는 `PUBLIC_DATA_KEY` 설정

### 출력

- `Collect/raw/seongsu_bus_stops_routes.json` – 정류장별 경유노선
- `Collect/raw/seongsu_bus_routes.txt` – 노선 ID 목록

### API 미사용 시

1단계 API 호출이 실패하면 **정적 노선 목록** `Collect/raw/seongsu_bus_routes_static.txt`를 사용합니다.  
성수역·뚝섬·서울숲 인근을 지나는 노선을 수동으로 정리한 목록입니다.

## 2단계: CardBusTimeNew 수집

### 실행

```bash
python Collect/collect_seoul_bus_seongsu.py -m 202601
```

### 필요 조건

- `.env`에 `SEOUL_OPEN_DATA_KEY` 설정 (서울열린데이터광장)

### 출력

- `Collect/raw/seoul_bus_{YYYYMM}_성수동노선_정류장별_시간대별_승하차.csv`

### 노선 목록 우선순위

1. `--routes`로 지정한 파일
2. `Collect/raw/seongsu_bus_routes.txt` (1단계 결과)
3. `Collect/raw/seongsu_bus_routes_static.txt` (정적 목록)

### 성수동 정류장만 필터 (선택)

1단계 JSON이 있을 때 성수동 정류장만 남기려면:

```bash
python Collect/collect_seoul_bus_seongsu.py -m 202601 --filter-stops Collect/raw/seongsu_bus_stops_routes.json
```

## 데이터 구조

| 컬럼 | 설명 |
|------|------|
| use_ym | 사용년월 (YYYYMM) |
| rte_no | 노선번호 |
| rte_nm | 노선명 |
| stops_id | 표준버스정류장ID |
| stops_ars_no | ARS번호 |
| stops_nm | 정류장명 |
| hour | 시 (00~23) |
| rideNope | 승차 인원 |
| gffNope | 하차 인원 |

## 참고

- CardBusTimeNew는 **월별** 데이터 (매월 5일 전월 데이터 갱신)
- sample 키 사용 시 노선당 5정류장만 수집됨
- 정상 인증키 사용 시 노선 전체 정류장 수집 가능
